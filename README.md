# minishell

### To clone:
git clone --recurse-submodules **GIT_URL**

### To branch:
git checkout -b *branch_name*
<br />
git add *files*
<br />
git commit -m *message*
<br />
git push --set-upstream origin *branch_name*

### To merge:
open pull request on github
<br />
merge commits on github
<br />
git branch -M main
<br />
git pull
